## 💻 Projeto

Projeto que armazena uma lista de tarefas.
Desenvolvido com Jetpack Compose, Firestore Database. 

## 🚀 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- Jetpack Compose
- Firestore Database.
